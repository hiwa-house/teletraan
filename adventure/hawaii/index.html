    • T-Mobile (Experience Beyond/More plan) phone #
    • View MacOS WiFi interface info: Option-click WiFi icon in the menu bar
    • Safari Developer Mode (Settings > Advanced > Show Develop) and set user agent to iPhone/iOS
    • MAC Spoofing
      1. Record phone MAC (9E:27:5E:2D:06:56)
      2. Turn off phone WiFi
      3. Disconnect from WiFi network, then turn off laptop WiFi
      4. sudo /System/Library/PrivateFrameworks/Apple80211.framework/Versions/Current/Resources/airport -z
      5. sudo ifconfig en0 ether AA:BB:CC:AA:BB:CC
      6. sudo ifconfig en0 lladdr AA:BB:CC:AA:BB:CC
      7. Turn on laptop WiFi, then connect to WiFi network
